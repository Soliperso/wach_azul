# Product Requirements Document

## Phase 0 Goal

Create the first runnable app shell and a vertical slice that proves the learning loop.

## M1 Vertical Slice

Splash -> Onboarding -> Family Village -> Tap Yemma -> Learn "Yemma" -> Listen & Match challenge -> XP -> Collectible -> Azul celebration -> come back tomorrow.

## Success Signal

A child can create a profile, tap Yemma in Family Village, learn the word Yemma, answer one listening match challenge, earn XP, unlock a collectible, and feel invited to return tomorrow.
