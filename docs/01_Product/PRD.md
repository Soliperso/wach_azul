# Product Requirements Document

## Phase 0 Goal

Create the first runnable app shell and a vertical slice that proves the learning loop.

## Vertical Slice

Splash -> Onboarding -> Family Village -> One word -> One challenge -> XP -> Collectible -> Azul celebration.

## Success Signal

A child can create a profile, learn Azul, answer one challenge, earn XP, collect a reward, and feel invited to return tomorrow.
