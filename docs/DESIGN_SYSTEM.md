# Wach? Azul! - Design System v1.0

## Purpose
The design system ensures every screen in Wach? Azul! feels magical, warm, clear, and consistent.

## Design Keywords
- Magical
- Warm
- Calm
- Playful
- Premium
- Child-safe
- Cultural
- Storybook-like

## Visual Direction
Wach? Azul! should feel like an animated storybook adventure, not a generic education app.

## Core Rule
Every visual choice should help the child understand, feel safe, or feel wonder.

## Colors

### Sky Blue
Used for calm backgrounds and open space.

### Sun Gold
Used for rewards, progress, warmth, and celebration.

### Clay Orange
Used for cultural warmth and primary actions.

### Leaf Green
Used for success, nature, and gentle positive feedback.

### Night Brown
Used for readable text and grounding.

## Typography
Use rounded, friendly, readable typography.

Recommended:
- Nunito
- Baloo 2
- Fredoka

## Buttons
Buttons should be:
- Large
- Rounded
- High contrast
- Easy to tap
- Animated gently on press

## Cards
Cards should feel like soft storybook panels.

Use:
- Rounded corners
- Warm shadows
- Spacious layout
- Clear icon/illustration area

## Animation
Animation should guide attention, not distract.

Use animation for:
- Azul greeting
- tap targets
- reward moments
- progress feedback
- world ambience

Avoid:
- excessive bouncing
- fast flashing
- cluttered motion

## Accessibility
A child who cannot read yet should still know what to do.

Use:
- visual cues
- sound cues
- clear motion
- large tap targets
- simple layouts

## Performance
The app must feel smooth on older phones.

Prioritize:
- lightweight animations
- optimized images
- local-first interactions
- minimal overdraw

## M1 Design Goal
For M1, one child should complete:

Splash -> Onboarding -> Family Village -> Yemma -> Challenge -> Reward

and feel:
- curious
- safe
- proud
- excited to return tomorrow
