# Wach? Azul! Game Design Document

## Game Fantasy

Wach? Azul! is an animated storybook adventure where a child explores magical cultural worlds, learns heritage-language words, earns warm rewards, and returns tomorrow with pride.

## Core Loop

1. Explore
2. Learn
3. Play
4. Earn
5. Celebrate
6. Return tomorrow

## M1 World

Family Village.

Family Village is the first proof of the game. It should feel warm, safe, familiar, and magical.

## M1 Word

Yemma.

Yemma is a family word. It should be introduced with care, warmth, sound, and meaning rather than as a flashcard.

## M1 Challenge

Listen & Match.

The child hears or is prompted with the Yemma sound and chooses the matching word from a small set of options.

## M1 Reward

The child earns XP and unlocks one collectible: Woven Heart.

## Companion

Azul is the companion guide. Azul should make the child feel safe, encouraged, and proud.

## Design Rule

Screens should be assembled from reusable game components and game systems:

- Design System
- Game Components
- Game Engine
- Screens

Canonical extended game design details live in `docs/product/Game_Design_Document.md`.
