# App Assets

Production app assets live here.

```text
assets/
  fonts/
  icons/
  illustrations/
  audio/
  animations/
```

Read `../../docs/production/Assets_Pipeline.md` before adding assets.

Rules:

- Use lowercase snake_case names.
- Keep M1 local-only.
- Optimize for older phones.
- Register assets in `pubspec.yaml` only when used.
- Do not place editable source files in this folder.
