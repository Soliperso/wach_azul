# Wach? Azul! Roadmap

## Current Phase

Phase 0 - Production Foundation.

We are building the game studio before building the game.

## Phase 0 - Production Foundation

Goal: make the repository understandable, coherent, and reusable for AI-assisted production.

Deliverables:

1. Constitution
2. PRD
3. Roadmap
4. GDD
5. Design System
6. Flutter Architecture
7. Sprint 1 plan

Checkpoint: Can any new AI engineer understand the project in 30 minutes?

## Phase 1 - M1 Vertical Slice

Goal: build one complete, magical, local learning loop.

Scope:

Splash -> Onboarding -> Family Village -> Yemma -> Challenge -> Reward -> Return Tomorrow.

## Phase 2 - Progression

Goal: extend the proven loop without changing the foundation.

Potential scope:

- More family words.
- Reward history.
- More map nodes.
- Stronger adventure progression.

## Phase 3 - Family and Data

Goal: add networked and family features only after the local loop is excellent.

Potential scope:

- Supabase.
- Family recordings.
- Parent-visible progress.

## Phase 4 - Polish and Scale

Goal: improve accessibility, performance, content depth, and production readiness.

Potential scope:

- Parent dashboard.
- Accessibility passes.
- Optimization.
- Larger world and content expansion.

Canonical roadmap details live in `docs/product/Roadmap.md`.
